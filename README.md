# Rutong Personal Knowledge Base

This repo is the working home for Rutong's AI PMO, AI Eco, deck, script, and BI analysis knowledge.

## Operating Rule

- Edit locally first. Push to GitHub only after Rutong confirms the structure or output.
- Keep durable knowledge in `docs/`.
- Keep executable agent behavior in `skills/`.
- Keep links and source artifacts visible in `docs/resource-index.md`.
- Run a regular neatness scan so stale links, duplicated notes, and oversized skill files do not accumulate.
