# Training / Adoption Lines

## 低压力互动句

- 这个回答没有好坏，按照大家自己工作里常见的处理方式回答就行。
- 大家可以先说一个最直觉的想法，不需要是标准答案。
- 如果你现在还没跟上，也没关系，我们先把入口和第一步跑通。
- 这一步不是为了考大家，而是让大家知道以后自己遇到类似问题可以怎么拆。

## Phase 1 Training 主线

AI Agent 入门培训的主线：

1. AI Agent 是什么：会理解目标、规划步骤、调用工具完成任务的智能助手。
2. Prompt：决定 Agent 的思考方式和行为边界。
3. KB：决定 Agent 的知识储备和可信来源。
4. Tool：决定 Agent 的执行能力。
5. Seatalk Bot / Timer：让 Agent 进入日常工作场景。
6. Multi-Agent / Workflow：面向更复杂流程。

## Adoption 讲法

把“AI 很强”改成：

- 你现在有一个重复工作。
- 这个工作可以拆成 prompt / KB / tool。
- 我们先做最小可用版本。
- 用起来后再看 coverage、saving time 和可复用性。
