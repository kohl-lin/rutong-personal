## 典型检查点

- shop / item / warehouse / region grain 是否一致。
- B2B / non-B2B 是否由正确字段判断。
- `merchant_region`、`shipment_type`、stage logic 是否混淆。
- leadtime 表是否因为 grain 不齐导致 duplicate rows。
- output summary table 是否和 corrected forecast output 对齐。
